Grove UI Framework Guidelines Colors

Colors

텍스트 및 배경 명암비에 대한 WCAG 지침을 준수하고 아름다운 색상 팔레트를 사용합니다.

Default

  • Primary
  • Secondary
// Default design palette
$color-primary: #0F62FE;
$color-secondary: #FF5151;

Primary color는 Secondary color보다 사용 비율이 높아야 합니다.
이와 반대로 사용하면 사용자는 주요색에 대한 혼란이 생길 수 있습니다.

System

  • Red
  • Yellow
  • Green
  • Blue
  • disabled
// System design palette
$color-red: #D72222;
$color-yellow: #FFC107;
$color-green: #26A769;
$color-blue: #0036B1;
$color-disabled: #F7F7F7;

컬러의 의미는 문화에 의해서 다를 수 있으나 도로교통표지에 대한 비엔나 협약에서 다음과 같은 색상들의 상태 의미를 국제적으로 표준화시켰습니다.
디자인에 이러한 표준을 사용하여 사용자에게 직관적으로 시스템 상태를 보여줍니다.

· Red : 오류, 금지, 정지, 위험, 삭제, 불가, 필수

· Yellow : 주의, 경고

· Green : 안전, 진행, 계속, 성공

· Blue : 정보 활성화, 링크

· #F7F7F7 : disabled

$blue-darken3: #001D6C;
$blue-darken2: #002D9C;
$blue-darken1: #0043CE;

기본 컬러에서 밝고 어두운 컬러를 명도와 채도의 단계를 두어 사용합니다.
마우스 호버 색상을 인터렉션 컬러를 활용하면 사용자가 어느 위치에 마우스를 호버하고 있는지 알 수 있습니다.

Grayscale은 계층 구조를 구별하는 데 도움이 되도록 구성 요소 내에서 주로 사용됩니다. 회색 음영을 배경, 테두리 및 그림자로 사용하여 영역을 강조하거나 구분합니다.
일반 텍스트는 명암비가 4.5:1 이상이어야 합니다.
