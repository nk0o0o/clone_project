Grove UI Framework UI Elements Icons

Icons

디자인 요소의 역할이 아닌 사용자의 행동을 명확하게 함으로써 내용에 도움이 되는 아이콘을 사용합니다.
의도적인 목적과 의미가 없는 장식용 아이콘은 지양합니다.

Layout

  • 실제 아이콘이 들어가는 라이브 영역과 아이콘 주위의 여유 공간을 두기 위한 패딩으로 구성합니다.

Size

사용하는 곳에 따라서 기본으로 16px, 24px, 32px, 48px 의 아이콘을 사용합니다.

· 16px : 최소 크기로 텍스트와 함께 사용하는 것을 권장합니다.

· 24px : 기본 크기로 버튼, 도움말 등에 사용합니다.

· 32px : 명확하게 정보 인지가 필요한 경우 사용합니다.

· 48px : 디테일한 묘사가 필요한 경우 사용합니다.

Linear

아이콘 이름을 복사한 후 class 이름으로 지정하세요. 새로운 아이콘을 추가하거나 기존 아이콘을 변경할 수도 있습니다.
