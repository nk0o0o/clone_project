Typography

타이포그래피를 사용하여 디자인과 콘텐츠를 최대한 명확하고 효율적으로 표현합니다.

Typeface

  • Korean
    //Default font
    $font-noto: "Noto Sans KR", sans-serif;
    .font_noto{font-family: $font-noto;}
    .font_kr{font-family: $font-noto;}

    //Embedded font
    $font-aritadotum: "AritaDotum", sans-serif;
    $font-lgsmha: "LGSmHa", sans-serif;

    .font_aritadotum{font-family: $font-aritadotum;}
    .font_lgsmha{font-family: $font-lgsmha;}
    .font_SUIT{font-family: $font-SUIT;}
  • English
    //Default font
    $font-pop: "Poppins", sans-serif;
    .font_pop{font-family: $font-pop;}
    .font_en{font-family: $font-pop;}

    //Embedded font
    $font-aritasans: "AritaSans", sans-serif;
    $font-lgsmha: "LGSmHa", sans-serif;
    $font-APHQ: "APHQ", sans-serif;

    .font_sans{font-family: $font-sans;}
    .font_lgsmha{font-family: $font-lgsmha;}
    .font_lAPHQ{font-family: $font-APHQ;}
폰트는 내장된 웹 폰트를 사용합니다.
폰트 타입 코드를 변수에 저장하고 수정할 때는 변수의 값만 변경하여 간단하게 적용합니다.
