Grove UI Framework Guidelines Spacing

Spacing

간격은 디자인에서 중요하지만 종종 간과되는 부분입니다. 간격은 추측하지 않고 명확하게 제공합니다.

Spacing scale

code rem px Example
$sp01 0.2 2
$sp02 0.4 4
$sp03 0.8 8
$sp04 1.2 12
$sp05 1.6 16
$sp06 2.4 24
$sp07 3.2 32
$sp08 4 40
$sp09 4.8 48
$sp10 6.4 64
$sp11 8 80
$sp12 9.6 96
margin: $sp03;
margin: $sp03 $sp01;
margin: $sp07 0 $sp04 0;
margin-right: $sp05;

padding: $sp04;
padding: $sp05 $sp03;
padding: $sp07 $sp04 0 $sp04;

.spm_t_2 {margin-top: $sp01;}
.spm_r_2 {margin-right: $sp01;}
.spm_l_2 {margin-left: $sp01;}
.spm_b_2 {margin-bottom: $sp01;}
.spm_tr_2, .spm_rt_2 {margin-top: $sp01; margin-right:$sp01;}
.spm_tl_2, .spm_lt_2 {margin-top: $sp01; margin-left:$sp01;}
.spm_tb_2, .spm_bt_2 {margin-top: $sp01; margin-bottom:$sp01;}
.spm_rl_2, .spm_lr_2 {margin-right: $sp01; margin-left:$sp01;}
.spm_rb_2, .spm_br_2 {margin-right: $sp01; margin-bottom:$sp01;}
.spm_lb_2, .spm_bl_2 {margin-left: $sp01; margin-bottom:$sp01;}

Stacking

스태킹은 구성 요소 또는 항목 사이에 같은 거리를 만드는 간격입니다.
항목 사이에 얼마나 많은 공간이 있어야 하는지를 결정하기 위해 간격 척도를 사용하는 스택 구성 요소를 제공하며, 수평 및 수직 방향을 모두 지원합니다.
